# VR Hardware Status

### Confirmed Working

* HTC Vive - 
  [SteamVR for Linux Github](https://github.com/ValveSoftware/SteamVR-for-Linux)
  (development release)
* Oculus Rift development kits (DK1, DK2) - 
  [ArchWiki](https://wiki.archlinux.org/index.php/Oculus_Rift)
